Wie wird der RILE-Index berechnet?

Der RILE-Index (Right–Left) verortet einen Text auf einer eindimensionalen Links-Rechts-Achse. Er folgt der Logik des Manifesto Project (MARPOR): aus den klassifizierten Anteilen werden die einer „rechten" bzw. „linken" Position zugeordneten Codes summiert und gegeneinander verrechnet:

RILE = (Σ rechts − Σ links) / (Σ rechts + Σ links) × 100

Werte nahe 0 bedeuten ein ausgewogenes Profil, positive Werte eine eher marktliberal-konservative, negative eine eher sozial-progressive Tendenz. Verwendet wird die Standard-Kategorienliste nach Laver/Budge.
